# droneschool リポジトリ共通ルール(AI コーディングエージェント向け)

このリポジトリはドローンエンジニア養成塾の教材リポジトリです。受講生は
`workshop/<期>/<氏名>/` 配下の自分のワークフォルダだけで作業します。

## 作業範囲(最重要)

- ファイルの作成・編集・削除は、**エージェントを起動したフォルダ(自分のワークフォルダ)の配下だけ**で行うこと。
- 次の場所は**読み取り専用**。参照やコピー元にするのは可、変更・削除は不可:
  - `webapp-blueos/` … 教材の完成版アプリと生成プロンプト
  - `.claude/` … Agent Skill・設定・ガードレール
  - 他の受講生の `workshop/` フォルダ
  - リポジトリ直下のファイル・上記以外の教材フォルダ(`pymavlink_scripts/` など)

## 禁止コマンド

以下は実行しない。必要な場合は、実行せずコマンドを提示してユーザー自身に実行してもらうこと。

- ワークフォルダ外を対象とする `rm` / `mv` などの削除・移動
- `git reset --hard` / `git clean` / `git checkout -- <file>` / `git restore` などの変更取り消し系
- `git push --force`
- `sudo` を伴うシステム変更

## Git 運用

- コミット・プッシュは、ユーザーが明示的に依頼したときのみ行う。
- コミット前に `git status` を確認し、変更が自分のワークフォルダ内に収まっていることを確認する。
- 課題提出の PR には `workshop/<期>/<氏名>/` 配下以外の変更を含めない(CI でチェックされます)。
