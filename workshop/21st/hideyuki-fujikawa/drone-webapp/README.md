# Drone Web App

## 概要

ドローンを Web ブラウザから操作できる最小構成のアプリケーションです。
FastAPI + WebSocket のバックエンドが pymavlink で機体（SITL）と通信し、
ブラウザ上で機体の状態をリアルタイムに確認しながら、アーム・離陸・着陸・
モード変更・指定座標への移動（GoTo）を実行できます。

## 機能一覧

- 機体への接続（接続ボタン契機。ページを開いただけでは接続しない）
- アーム / ディスアーム
- 離陸（目標高度を指定）
- 着陸
- GoTo（緯度・経度・高度を指定して移動）
- フライトモード変更（GUIDED / AUTO / RTL / LOITER / STABILIZE）
- リアルタイム状態表示（接続・アーム・モード・緯度・経度・高度・ヘディング）
- 地図表示（Leaflet + OpenStreetMap）: 機体マーカーの追従、飛行軌跡の描画

## 技術スタック

| 区分 | 技術 |
| --- | --- |
| バックエンド | Python / FastAPI / uvicorn |
| リアルタイム通信 | WebSocket |
| MAVLink 通信 | pymavlink |
| フロントエンド | HTML + JavaScript + CSS（ビルドツール不要） |
| 地図 | Leaflet + OpenStreetMap タイル |

## 前提条件

- Python 3.7 以上
- SITL（シミュレータ）が `tcp:127.0.0.1:5762` で待ち受けていること

## 起動手順

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --port 9999 --reload
```

ブラウザで `http://127.0.0.1:9999/` を開きます。

## 使い方

1. ページを開くと WebSocket でサーバーに自動接続されます（機体へはまだ接続されません）
2. 「接続」ボタンを押すと機体への MAVLink 接続が始まります
3. 接続後、ステータスパネルと地図がリアルタイムに更新されます
4. 「アーム」→ 離陸高度を入力して「離陸」で離陸します
5. GoTo に緯度・経度・高度を入力して「移動」で指定地点へ移動します
6. モードを選択して「設定」でフライトモードを変更します（RTL で帰還など）
7. 「着陸」で着陸します
