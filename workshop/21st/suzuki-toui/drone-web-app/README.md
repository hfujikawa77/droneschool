# ドローンWeb制御アプリケーション

## 概要

このアプリケーションは、Webブラウザを使用してドローンをリアルタイムに制御・監視するための最小構成アプリケーションです。FastAPI、WebSocket、pymavlinkを使用して、ローカルで動作します。

## 機能一覧

- **リアルタイム接続管理**: MAVLink経由でドローンに接続・切断
- **機体制御**: アーム/ディスアーム、離陸、着陸、GoTo移動、モード変更
- **リアルタイム状態表示**: 接続状態、アーム状態、フライトモード、位置情報（緯度・経度・高度）、ヘディング
- **地図表示**: Leaflet + OpenStreetMapタイルで機体位置を可視化
- **飛行軌跡表示**: ポリラインで飛行経路を記録
- **レスポンシブUI**: PC、タブレット、スマートフォンに対応

## 技術スタック

- **バックエンド**: Python 3.7+ / FastAPI / uvicorn
- **フロントエンド**: HTML5 / JavaScript / CSS3
- **MAVLink通信**: pymavlink
- **リアルタイム通信**: WebSocket
- **地図ライブラリ**: Leaflet

## 前提条件

1. **Python 3.7以上**がインストールされていること
2. **SITL（シミュレータ）** が `tcp:127.0.0.1:5762` で待ち受けていること
   - ArduPilotのSITLシミュレータを起動してください

## 起動手順

### 1. 依存関係のインストール

```bash
cd backend
pip install -r requirements.txt
```

### 2. バックエンドサーバーの起動

```bash
uvicorn main:app --port 9999 --reload
```

### 3. ブラウザでアクセス

ブラウザを開いて `http://127.0.0.1:9999/` にアクセスしてください。

## 使い方

### 基本的な操作フロー

1. **機体に接続**: 「機体に接続」ボタンをクリック
   - SITL（`tcp:127.0.0.1:5762`）に接続を試みます
   - 接続成功時、状態が「接続中」に変わります

2. **アーム**: 「アーム」ボタンをクリック
   - ドローンのモーターを起動準備します

3. **離陸**: 離陸高度を入力して「離陸」ボタンをクリック
   - 自動的にGUIDEDモードに切り替わって離陸します

4. **操作**:
   - **GoTo**: 目標の緯度・経度・高度を入力して「移動」ボタンをクリック
   - **モード変更**: ドロップダウンからモード（GUIDED/AUTO/RTL/LOITER/STABILIZE）を選択して「設定」ボタンをクリック

5. **着陸**: 「着陸」ボタンをクリック

6. **ディスアーム**: 着陸後に「ディスアーム」ボタンをクリック

### UI パネル説明

- **ステータスパネル** (左): リアルタイムの機体状態を表示
- **コントロールパネル** (中央): 各種操作ボタンとコマンド入力フォーム
- **地図** (右): 機体位置をマーカーで表示、飛行軌跡をポリラインで表示

## トラブルシューティング

| 症状 | 原因 | 対処法 |
| --- | --- | --- |
| 「WebSocket接続できません」 | バックエンドが起動していない | `uvicorn` コマンドを実行してください |
| 「機体に接続できません」 | SITL が起動していない | ArduPilotのSITLを `tcp:127.0.0.1:5762` で起動してください |
| 位置情報が更新されない | MAVLinkデータストリームが送信されていない | SITL または 実機のテレメトリー設定を確認してください |
| ボタンが反応しない | WebSocket接続が切断されている | ページをリロードして再接続してください |
