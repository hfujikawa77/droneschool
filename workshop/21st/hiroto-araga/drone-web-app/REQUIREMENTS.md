# 要件定義

## 概要
ブラウザからドローンの基本操作を行うための Web アプリケーションです。

## 目的
CLI ベースの操作を Web UI に置き換え、ローカル環境で簡単に試せるようにします。

## ターゲットユーザー
- ドローン制御の学習者
- ローカル SITL で動作確認したい開発者

## 機能要件
- MAVLink 接続
- アーム / ディスアーム
- 離陸 / 着陸
- GoTo
- モード変更
- 地図表示

## 非機能要件
- FastAPI + WebSocket でリアルタイム通信
- モバイル幅でも操作可能
- 未接続時でも UI が壊れない

## 将来拡張案
- 追加コマンドの実装
- 3D マップやビデオストリーム連携
- 認証付き運用
